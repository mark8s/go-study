### 系统os包


#### 介绍
os包中提供了操作系统函数的接口，是一个比较重要的包。它的作用主要是在服务器上进行系统的基本操作，如文件操作、目录操作、执行命令、信号与中断、进程、系统状态等等。

#### 常用函数

- Hostname(): 获取当前主机名
- Getpid(): 返回调用者所在进程的进程ID
- Getppid: 返回调用者所在进程的父进程的进程ID
- Exit(): 让程序以状态码code退出。状态码0表示成功，非0表示出错。 code取值范围： [0,125]
